# Sohrab Ahmad Siddiqui — Portfolio

Personal portfolio website showcasing projects, skills and contact details.

> Live preview: open `index.html` in a browser (or see GitHub Pages if deployed).

## Overview
This repository contains a single-page portfolio built with plain HTML, CSS and minimal JavaScript — a modern dark/light themed design with particle background, responsive layout and contact form mailto integration.

Files of interest:
- `index.html` — main page markup. :contentReference[oaicite:0]{index=0}  
- `style.css` — styles and responsive rules. :contentReference[oaicite:1]{index=1}

## Features
- Hero section with animated particle background and theme toggle
- Sections: About, Skills, Projects, Experience, Contact
- Responsive layout and subtle animations
- Contact form opens default email client (mailto)
- Social links to LinkedIn, GitHub, LeetCode, GeeksforGeeks, Instagram, X

## Tech stack
- HTML5
- CSS3 (modern layout, animations)
- Vanilla JavaScript (particles, theme toggle, smooth scrolling)
